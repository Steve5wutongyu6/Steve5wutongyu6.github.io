BetterReadingList

一个更强大的Chrome阅读列表

厌倦了简易的Chrome阅读列表？试试这个！无缝对接Chrome原生阅读列表，在原生阅读列表基础上增加了搜索功能和两种不同的排序方式。让你更快速地在阅读列表中找到你收藏的文章！

功能

  • 搜索
    • 支持标题和网址的模糊搜索
    • 实时搜索结果展示
    • 支持按标签筛选
  • 排序
    • 按创建时间排序
    • 按更新时间排序

安装

  1. 从Chrome网上应用店安装
  2. 从Microsoft Edge网上应用店安装
  3. 从GitHub下载源码安装

使用说明

  1. 点击Chrome工具栏中的扩展图标即可打开
  2. 点击添加当前标签页，将当前标签页添加到阅读列表
  3. 在阅读列表中管理您的阅读列表

BetterReadingList

A more powerful Chrome reading list

Tired of the simple Chrome reading list? Try this one! Seamlessly integrated with Chrome's native reading list, it adds search functionality and two different sorting methods. Find your saved articles faster!

Features

  • Search
    • Fuzzy search for titles and URLs
    • Real-time search results
    • Filter by tags
  • Sorting
    • Sort by creation time
    • Sort by update time

Installation

  1. Install from Chrome Web Store
  2. Install from Microsoft Edge Add-ons Store
  3. Download and install from GitHub

Usage

  1. Click the extension icon in the Chrome toolbar to open
  2. Click "Add Current Tab" to add the current tab to the reading list
  3. Manage your reading list in the extension

Privacy Policy

BetterReadingList values your privacy. This privacy policy explains how we collect, use, and protect your information.

Information Collection

This extension:

  • Does not collect any personally identifiable information
  • Does not track your browsing history
  • Does not store your personal data
  • Only accesses the URLs and titles of pages you add to the reading list
  • All data is stored locally in your browser

Permissions

This extension requests the following permissions:

  • readingList: To access and manage Chrome's reading list
  • sidePanel: To display the reading list in the sidebar
  • contextMenus: To add right-click menu options
  • tabs: To get information about the current tab

These permissions are only used for the basic functionality of the extension and will not be used for any other purposes.

Data Storage

  • All data is stored locally in your browser
  • No data is uploaded to external servers
  • You can delete all data at any time through Chrome's settings

Updates

We reserve the right to update this privacy policy. If there are significant changes, we will notify you through the extension update.

Contact Us

If you have any questions about this privacy policy, please contact us by submitting an Issue on GitHub.
